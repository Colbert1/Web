Hockey Sur Glace

Le hockey sur glace est un sport collectif olympique qui se déroule sur une patinoire.

C'est l'un des seuls sports où les joueurs ont le droit de se déplacer derrière les cages.

Durant un match, deux équipes de six joueurs (dont un gardien de but) s’affrontent.

Le but est d’envoyer un palet (ou rondelle) dans le but adverse, grâce à une crosse (ou bâton de hockey).

Chaque joueur possède une crosse et se déplace sur le terrain avec des patins.

Le hockey sur glace a été inventé dans l'Est du Canada dans le début de 19e siècle, où il est toujours très populaire.

Il apparaît pour la première fois en tant que sport olympique aux Jeux olympiques d’été de 1920 à Anvers.

À partir de 1924, il est pratiqué aux Jeux olympiques d'hiver (avant cette date, seuls les Jeux olympiques d'été existaient).
